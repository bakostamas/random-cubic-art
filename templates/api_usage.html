Random cubic image generator

Available API functions

  • /api/get-clean-image
  • /api/get-noisy-image

Optional parameters

  • length: Length of the image in pixels (default: 500)
  • height: Height of the image in pixels (default: 500)
  • style: Color style code to apply - valid codes: [0-24] (default: randomly chosen)
  • iter: Number of iterations - the number of rectangles on result image (default: 200)

Examples

  • /api/get-clean-image?length=1024&height=768
  • /api/get-noisy-image?style=2
  • /api/get-noisy-image?iter=10
